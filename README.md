# Photo Board Android app
사진과 제목, 내용 등을 작성하여 서버에 저장하고 이를 받아 화면에 출력하는 앱입니다.
서버는 스프링부트 프레임워크를 이용하여 구현하였습니다.
안드로이드 앱에서 http 연결을 통해 서버와 접속하여 사진 및 JSON 데이터를 송수신합니다.
